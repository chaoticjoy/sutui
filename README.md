sutui PHP SDK
=====

PHP SDK for sutui

基于速推微信公众号的PHP SDK，速推微信公众号：sutui

速推是基于微信公众号的消息推送中心及控制台。可以帮助开发者安全快速地实现例如服务器或业务异常告警，远程控制服务器等需求。

速推不局限不以上用法，可以通过速推方便的推送文字、图文、音乐、视频等消息，也可以通过简单的微信命令完成远程操作，有想象力的开发者还可以把速推应用到更多场合中。
